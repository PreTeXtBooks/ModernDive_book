    Section 10.3 Simulation-based inference for simple linear regression

    In this section, we’ll use the simulation-based methods you previously learned in Chapter 8 and Chapter 9 to recreate the values in the regression table. In particular, we’ll use the infer package workflow to
    🔗
    • Construct a 95% confidence interval for the population slope \(\beta_1\) using bootstrap resampling with replacement.
      🔗
      🔗
    • Conduct a hypothesis test of \(H_0: \beta_1 = 0\) versus \(H_A: \beta_1 \neq 0\) using a permutation test.
      🔗
      🔗

    Subsection 10.3.1 Confidence intervals for the population slope using infer

    We’ll construct a 95% confidence interval for \(\beta_1\) using the infer workflow. Specifically, we’ll first construct the bootstrap distribution for the fitted slope \(b_1\) using our single sample of eruptions:
    🔗
    1. specify() the variables of interest in old_faithful_2024 with the formula: waiting ~ duration.
      🔗
      🔗
    2. generate() replicates by using bootstrap resampling with replacement from the original sample. We generate reps = 1000 replicates using type = "bootstrap".
      🔗
      🔗
    3. calculate() the summary statistic of interest: the fitted slope \(b_1\text{.}\)
      🔗
      🔗
    It is important to note in this case that the bootstrapping with replacement is done row-by-row. Thus, the original pairs of waiting and duration values are always kept together, but different pairs may be resampled multiple times. The resulting confidence interval will denote a range of plausible values for the unknown population slope \(\beta_1\text{.}\)
    🔗
    bootstrap_distn_slope <- old_faithful_2024 |>
  specify(formula = waiting ~ duration) |>
  generate(reps = 1000, type = "bootstrap") |>
  calculate(stat = "slope")
bootstrap_distn_slope
    Observe how we have 1000 values of the bootstrapped slope \(b_1\) in the stat column. We visualize these bootstrapped values:
    🔗
    visualize(bootstrap_distn_slope)
    Observe how the bootstrap distribution is roughly bell-shaped. Recall that the shape of the bootstrap distribution of \(b_1\) closely approximates the shape of the sampling distribution of \(b_1\text{.}\)
    🔗

    Subsubsection 10.3.1.1 Percentile method

    First, let’s compute the 95% confidence interval for \(\beta_1\) using the percentile method. We identify the 2.5th and 97.5th percentiles which include the middle 95% of values. Recall that this method does not require the bootstrap distribution to be normally shaped.
    🔗
    percentile_ci <- bootstrap_distn_slope |>
  get_confidence_interval(type = "percentile", level = 0.95)
percentile_ci
    🔗

    Subsubsection 10.3.1.2 Standard error method

    Since the bootstrap distribution appears to be roughly bell-shaped, we can also construct a 95% confidence interval for \(\beta_1\) using the standard error method. We first compute the fitted slope \(b_1\) using the infer pipeline with the generate() step removed:
    🔗
    observed_slope <- old_faithful_2024 |>
  specify(waiting ~ duration) |>
  calculate(stat = "slope")
observed_slope
    We then use get_ci() with level = 0.95 and set the point_estimate argument to the observed_slope:
    🔗
    se_ci <- bootstrap_distn_slope |>
  get_ci(level = 0.95, type = "se", point_estimate = observed_slope)
se_ci
    The resulting standard error-based 95% confidence interval for \(\beta_1\) is slightly different than the percentile-based confidence interval. Note that neither of these confidence intervals contains 0 and both are entirely located above 0, suggesting a meaningful positive relationship between waiting times and duration for Old Faithful eruptions.
    🔗
    🔗
    🔗

    Subsection 10.3.2 Hypothesis test for population slope using infer

    Let’s conduct a hypothesis test of \(H_0: \beta_1 = 0\) vs. \(H_A: \beta_1 \neq 0\text{.}\) Recall that if \(\beta_1 = 0\text{,}\) there is no relationship between waiting time and duration. We can therefore shuffle/permute the waiting variable with no consequence.
    🔗
    We construct the null distribution of the fitted slope \(b_1\) by:
    🔗
    1. specify() the variables of interest with the formula: waiting ~ duration.
      🔗
      🔗
    2. hypothesize() the null hypothesis of independence.
      🔗
      🔗
    3. generate() reps = 1000 replicates by permuting/shuffling values using type = "permute".
      🔗
      🔗
    4. calculate() the test statistic: the fitted slope \(b_1\text{.}\)
      🔗
      🔗
    null_distn_slope <- old_faithful_2024 |>
  specify(waiting ~ duration) |>
  hypothesize(null = "independence") |>
  generate(reps = 1000, type = "permute") |>
  calculate(stat = "slope")
    Notice how it is centered at \(b_1 = 0\text{.}\) This is because in our hypothesized universe, there is no relationship between waiting and duration and so \(\beta_1 = 0\text{.}\) We obtain the observed slope:
    🔗
    b1 <- old_faithful_2024 |>
  specify(waiting ~ duration) |>
  calculate(stat = "slope")
b1
    We visualize the \(p\)-value by shading the null distribution and comparing to the observed test statistic:
    🔗
    visualize(null_distn_slope) +
  shade_p_value(obs_stat = b1, direction = "both")
    Since the observed fitted slope falls far to the right of this null distribution, we compute the numerical \(p\)-value:
    🔗
    null_distn_slope |>
  get_p_value(obs_stat = b1, direction = "both")
    The \(p\)-value is 0, matching the regression table. We therefore reject the null hypothesis \(H_0: \beta_1 = 0\) in favor of the alternative \(H_A: \beta_1 \neq 0\text{.}\) We have evidence that suggests there is a significant relationship between waiting time and duration values for eruptions of Old Faithful. When the conditions for inference are met and the null distribution has a bell shape, we expect similar results from simulation-based and theory-based approaches.
    🔗

    1.
    Repeat the inference but this time for the correlation coefficient instead of the slope. Note the implementation of stat = "correlation" in the calculate() function of the infer package.
    🔗
    Answer.
    Replace stat = "slope" with stat = "correlation" in both the bootstrap and null-distribution pipelines. The resulting confidence interval and \(p\)-value should be consistent with those obtained for the slope, since the two statistics carry equivalent information about the linear relationship.

    2.
    Why is it appropriate to use the bootstrap percentile method to construct a 95% confidence interval for the population slope \(\beta_1\) in the Old Faithful data?
    🔗
    1. Because it assumes the slope follows a perfect normal distribution.
      🔗
      🔗
    2. Because it relies on resampling the residuals instead of the original data points.
      🔗
      🔗
    3. Because it requires the original data to be uniformly distributed.
      🔗
      🔗
    4. Because it does not require the bootstrap distribution to be normally shaped.
      🔗
      🔗
    Answer.
    D. The percentile method does not require the bootstrap distribution to be normally shaped, making it applicable in a wider range of situations.
    🔗
    🔗
    🔗
    3.
    What is the role of the permutation test in the hypothesis testing for the population slope \(\beta_1\text{?}\)
    🔗
    1. It generates new samples to confirm the confidence interval boundaries.
      🔗
      🔗
    2. It assesses whether the observed slope could have occurred by chance under the null hypothesis of no relationship.
      🔗
      🔗
    3. It adjusts the sample size to reduce sampling variability.
      🔗
      🔗
    4. It ensures the residuals of the regression model are normally distributed.
      🔗
      🔗
    Answer.
    B. The permutation test shuffles the response variable to simulate a world where there is no relationship, and then measures how extreme the observed slope is relative to the resulting null distribution.
    🔗
    🔗
    🔗
    4.
    After generating a null distribution for the slope using infer, you find the \(p\)-value to be near 0. What does this indicate about the relationship between waiting and duration in the Old Faithful data?
    🔗
    1. There is no evidence of a relationship between waiting and duration.
      🔗
      🔗
    2. The observed slope is likely due to random variation under the null hypothesis.
      🔗
      🔗
    3. The observed slope is significantly different from zero, suggesting a meaningful relationship between waiting and duration.
      🔗
      🔗
    4. The null hypothesis cannot be rejected because the \(p\)-value is too small.
      🔗
      🔗
    Answer.
    C. A near-zero \(p\)-value means the observed slope is very unlikely under the null hypothesis, providing strong evidence of a meaningful relationship.
